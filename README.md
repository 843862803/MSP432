## 部分工程介绍
>`Mode_of_MSP432` CCS编译器使用的基于MSP432固件库的工程文件，下载后使用CCS直接导入即可使用固件库编程。

>`VS_MSP432_Mode` VS编译器使用的基于MSP432固件库的工程文件，使用VS打开工程后链接需要编写的.c文件到USER目录即可。
